#Linux 常用命令清单

## 文件操作

- pwd : 查看当前路径
- ls : 列出当前目录文件
- cd : 进入目录
- mkdir : 创建目录
- cp : 复制文件
- mv : 移动文件
- rm : 删除文件

## 文件查看:

- cat : 查看全部内容
- less : 分页查看
- head : 看前几行
- tail : 后几行

## 搜索

- grep : 查找关键词

## 权限

- chmod : 修改权限
- chown : 修改所有者

## 进程

- ps : 查看进程
  -top : 实时监控
- kill : 结束进程

## 网络

- ping : 测试网络
- curl : 请求接口
